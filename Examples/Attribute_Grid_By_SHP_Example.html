Starting Conditions



This displays an example grid with an irregular shapefile overlaid onto it.

The grid starts with initial values of 1 for each cell, which we want to modify if they intersect with the shapefile




Function Example



In this case the user wants to replace the 1 values with 3. Values should be replaced if cells have greater than 10% (partial_fact) of their area overlapped by the shapefile, with their new value scaled by the amount overlapped (scale_by_intersected_area). For example a cell 10% overlapped would have 10% of the distance between 1 and 3 (0.2) added to its starting value for a final value of 1.2.

# -------------------------------------------------------------------------------------
# Run function
Attribute_Grid_By_SHP(out_dir = '',
                      out_name = '',
                      shapefile = polygon,
                      model_grid = grid,
                      hru_id_column = 1,
                      replacement_value = 3,
                      starting_values = rep(1,100),
                      export_as_grid = TRUE,
                      export_long_format = TRUE,
                      is_raster = FALSE,
                      grid_dims = c(10,10),
                      null_value = 0,
                      engulf = FALSE,
                      partial_fact = 10,
                      scale_by_intersected_area = TRUE,
                      example = TRUE)

Function Example 2



The function can also take in a raster as the starting values. This time we will set the replacement value to 100, the starting value to the raster we generated above, and cells to only have their value replaced when they are fully engulfed (to make the visualization more obvious). In this case the user wants to replace the 1 values with 3. Values should be replaced if cells have greater than 10% (partial_fact) of their area overlapped by the shapefile, with their new value scaled by the amount overlapped (scale_by_intersected_area). For example a cell 10% overlapped would have 10% of the distance between 1 and 3 (0.2) added to its starting value for a final value of 1.2.
